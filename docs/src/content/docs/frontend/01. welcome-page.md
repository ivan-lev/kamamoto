## Методологии

- BEM
- DRY
- Desktop-first

## Используемые технологии и пакеты

### Основные

- `vite` для развития и сборки проекта
- `react 19` - базовая библиотека для реализации SPA
- `react compiler` для автоматической генерации обёрток useMemo и useCallback
- `react-router` для реализации роутинга
- `@reduxjs/toolkit` (RTK) + `react-redux` для хранения состояния переменных и работы с ними (запросы к api пока не переведены на RTK Query, ходят напрямую)

### Дополнительные

- `swiper` для создания слайдеров
- `motion` для анимирования компонентов
- `leaflet` / `react-leaflet` / `react-leaflet-markercluster` для карты гончарных центров
- `qrcode.react` для генерации QR-кодов
- `react-scroll-to-top` для кнопки прокрутки наверх

### Разное

- `html-react-parser` для парсинга html-информацииб хранящейся в бд
- `@emailjs/browser` для отправки сообщений из раздела "контакты"
- `@fontsource/raleway` для подключения шрифта

## Скрипты

- `npm run build` — создание рабочей сборки для размещения на сайте
- `npm run dev` — запуск проекта в режиме разработчика
- `npm run lint` — проверка кода eslint
- `npm run stylelint` — проверка и автофикс стилей (scss)
