## Раздел о мастерах ЛНТ (Living National Treasure)
- 2.5.1 Сделал универсальный компонент статьи, переиспользуемый для разных категорий
- 2.5.2 Добавил на бэке отдельный эндпоинт `GET /potters/lnt` для подборки гончаров категории LNT
- 2.5.3 Добавил на фронте страницы списка (`LNTPotters`) и отдельной статьи (`LNTPotter`) мастеров LNT
- 2.5.4 Сделал универсальный компонент в админ-панели для управления статьями мастеров

## Другие изменения
- Стор теперь подключается лениво — только если на странице реально есть компоненты, которым он нужен
- Почистил проект от неиспользуемых компонентов и пакетов
